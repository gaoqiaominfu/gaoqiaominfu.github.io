通过 Hexo 插件插入音乐/视频
这里需要使用两个播放器插件

hexo-tag-aplayer:https://github.com/grzhan/hexo-tag-aplayer#upstream-issue

hexo-tag-dplayer:https://github.com/NextMoe/hexo-tag-dplayer
两款插件基于 DIYgod 编写的 html5 播放器 APlayer 和 DPlayer 开发。

首先安装两款插件

打开 Git Shell，切换到 Hexo 目录下，运行如下两条命令

1
2
3
npm install hexo-tag-dplayer –save
npm install hexo-tag-aplayer –save

安装到最后 npm 可能会报错，忽视即可。

安装成功后，在 Markdown 文档中添加 APlayer 和 DPlayer 标签即可
