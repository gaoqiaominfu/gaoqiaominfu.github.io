Nexus 系列手机及平板产品的解锁、反解锁、ROOT等功能的实现非 Nexus root toolkit 莫属，它是Nexus系列产品的专属软件。最新版本的Nexus root toolkit 1.7.3 支持对Nexus4 / Nexus7 等最新系统 Andriod 4.3 的 root /解锁、刷机、备份、安装ADB驱动、刷recovery等操作，并且成功率最高。

使用 Nexus root toolkit 1.7.3 进行 Andrioid 4.3 root 图文教程：
Nexus root toolkit 安装完毕后，会弹出如下图所示的对话框，意即如何开启Nexus设备的USB调试；如果你已开启USB调试，点击确定即可，如果没有，而且不知道如何开启请按提示操作，或参考《Google Nexus 4 常见问题解决方案汇总》中的第一项。

一、选择设备：
成功安装Nexus root toolkit后，分两步选择你的设备型号，以安装了Android 4.3 的Nexus4为例：在上面的选项中选择Nexus4（LG E960）;选择Nexus4的具体版本号，此版本号信息可以在手机的设备-关于手机-版本号中找到；选择后点击右下角的Apply按钮即可。

二、安装驱动：
点击下图即图1中的1处，打开“Full Driver Configuration Guide”的驱动安装界面。

未安装驱动的用户直接点击“Full Driver Configuration Guide”界面中”Step2″，如下图所示，按软件界面中推荐的驱动解决方案#1(你的有可能是另外三个)，点击“Driver Solution #1”下面的“Universa Drivers”按钮安装驱动。

驱动安装完成后，为了验证驱动是否可用，先将手机打开USB调试，连接至电脑，然后进入“Step3”，点击“Full Driver Test”进行驱动验证。如果驱动验证成功会自动重启你的手机（过程中千万不要断开手机连接！）并显示驱动验证成功对话框。


如果驱动没有验证成功。动手能力差的同学可以通过安装腾讯手机管家2.0（点击下载）并与手机连接来解决驱动安装问题。

三、Root 手机：
请备份你的数据！再进行以下操作！Root之前，需要点击图1中2处的“Unlock”进行解锁，解锁操作会清空手机内所有数据！在解锁完成后，回到Nexus root toolkit用户界面（如图1），点击3处的 Root 按提示进行操作即可！如果在Root过程中提示需要下载对应的Root文件和Recoveryy,请允许下载。

四、Nexus root toolkit 1.7.3 的更多功能：
你在完成前三个步骤后，手机ROOT已完成。而Nexus root toolkit 1.6.6的功能远远不止这些，它还支持刷入刷入Recovery；Nexus4 /Nexus7 等刷回官方ROM并进行反Root等操作，有兴趣的朋友可以自行试之。

    recovery中文是恢复的意思，就是一个刷机的工程界面，在这个界面你可以直接用sd卡上的zip格式rom升级系统或备份系统。
    Recovery中常见英文的意思：（粗体部分就是常说的双wipe、3wipe、4wipe）
    -Reboot system now————重启
    -Wipe Data/Factory reset————清除内存数据和缓存数据
    -Wipe Cache Partition————清除cache分区内数据
    -Wipe Dalvik Cache————清空虚拟机缓存（可以解决一些程序FC问题）
    -Wipe Battery Stats————清空电池调试记录数据
    -install zip from sdcard————刷入sd卡中的.zip刷机包
    -install zip from adbsideload————通过adb sideload 模式从电脑端刷.zip 包
    -apply update from sdcard ————安装sd卡上的升级包
    -Backup/Restore————备份和还原系统（相当于电脑的ghost一键备份、还原）
    -backup ————备份系统
    -restore ————还原系统
    -advanced restore ————高级还原，一般用不到
    -mounts and storage ————挂载存储选项，通常用作U盘
    -advance ————高级功能
    -set permissions————修复Root权限（出现异常的时候可以修复权限）
    -power off ————关机
